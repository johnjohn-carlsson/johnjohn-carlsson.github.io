Ever since my much more coding experienced classmate showed me his terminal themed layout for his website,
I have been dreaming of creating something similar for myself. Well here it is! 

I give you; the VS Code themed version of john-john.nu.
It's really kind of a weird setup for a website, let me explain.

There is only one .html page that consists of the background image, an input box and an output box.
Javascript checks the input command when sent, if it recognizes the input text as an accepted command
in a list of commands, it loads the corresponding .txt file into the output field.
So yeah, that means I did write some HTML code in .txt files, it wasn't great.. good practice though!

You can try out the terminal version of this webpage <a href="https://johnjohn-carlsson.github.io/terminal/" target="_blank">here</a>.
