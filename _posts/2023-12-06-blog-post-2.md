WeatherSender.
======

Today I uploaded a new small python project to GitHub,
I call it WeatherSender.
You can find it [here](https://github.com/johnjohn-carlsson/WeatherSender).

The original idea was to get an automatic weather forecast every 
morning to my phone in the form of a text message, but I quickly 
realised that I would need some sort of cloud or server setup in order 
for this to work.

So until I figure that last step out, the project will act as a 
fully functioning script that sends the report when ran.

.. maybe I can convince someone to run the program on their computer
with the forecast going to my phone each morning! Would that still
count as an automatic system..?
